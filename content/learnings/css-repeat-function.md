CSS Grid introduces a handy `repeat()` function that works with both CSS
Grid properties grid-template-columns AND grid-template-rows.

It supports units of:

* `px`
* `percent`
* fractional `fr`'s
* `auto`
* `min-content` (?)
* `max-content` (?)
* `minmax` (?)
* `fit-content(200px)` (?)
